## Как читать уроки

Каждый урок объясняет:
1. **Зачем** — почему это нужно, какую проблему решает
2. **Что** — теория, объяснение концепций
3. **Как** — примеры кода из нашего проекта
4. **Задание** — что попробовать самостоятельно

Начинайте с урока 01 и идите по порядку. Каждый урок опирается на предыдущие.
