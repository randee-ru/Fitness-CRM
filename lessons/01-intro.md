# Урок 01 — Введение и инструменты

## Что мы строим и зачем

SportMax ERP — это система управления фитнес-клубом. Представьте: каждый день через клуб проходят сотни людей. Нужно знать:
- кто пришёл и вышел
- у кого есть абонемент, у кого нет
- кто заморозил абонемент
- кто записался на йогу в среду
- сколько денег поступило за день
- какой менеджер продал больше всех

Всё это — одна система, SportMax ERP.

Мы строим её с нуля, используя современные технологии. Это **реальный коммерческий продукт**, а не учебный проект.

---

## Что такое Full Stack разработка

Наш проект состоит из двух частей:

**Фронтенд (Frontend)** = то, что видит пользователь в браузере
- Кнопки, таблицы, формы, графики
- Написан на Next.js + React + TypeScript
- Работает в папке `apps/web/`

**Бэкенд (Backend)** = невидимая часть, работает на сервере
- Хранит и обрабатывает данные
- Отвечает на запросы фронтенда
- Написан на NestJS + TypeScript
- Работает в папке `apps/api/`
- Данные хранит в PostgreSQL (база данных)

Вместе это называется **Full Stack** разработка.

## Что нужно установить

### 1. Node.js — движок для JavaScript/TypeScript

Node.js позволяет запускать JavaScript не только в браузере, но и на сервере.

**Скачать:** https://nodejs.org/ (выбирайте LTS версию, сейчас это 20.x)

Проверить установку:
```bash
node --version   # должно показать v20.x.x
npm --version    # должно показать 10.x.x
```

### 2. Git — система контроля версий

Git отслеживает все изменения в коде. Позволяет работать в команде без конфликтов.

**Скачать:** https://git-scm.com/

Проверить:
```bash
git --version   # git version 2.x.x
```

### 3. Docker — контейнеры для баз данных

Docker позволяет запускать PostgreSQL и Redis на вашем компьютере в изолированных контейнерах. Не нужно устанавливать их вручную.

**Скачать:** https://www.docker.com/products/docker-desktop

Проверить:
```bash
docker --version         # Docker version 24.x.x
docker compose version   # Docker Compose version v2.x.x
```

### 4. VS Code — редактор кода

Самый популярный редактор для веб-разработки.

**Скачать:** https://code.visualstudio.com/

**Рекомендуемые расширения:**
- `Prisma` — подсветка схемы Prisma
- `ESLint` — проверка кода
- `Tailwind CSS IntelliSense` — автодополнение стилей
- `GitLens` — продвинутая работа с Git

## Первый запуск проекта

```bash
# 1. Скачать проект
git clone https://github.com/randee-ru/Fitness-CRM.git sportmax
cd sportmax

# 2. Запустить базы данных
docker compose -f docker-compose.dev.yml up -d

# 3. Установить зависимости
cd apps/api && npm install
cd ../web && npm install && cd ../..

# 4. Настроить базу данных
cd apps/api
cp ../../.env.example .env
npx prisma migrate dev
npm run db:seed

# 5. Запустить API (в новом терминале)
npm run dev

# 6. Запустить Web (в ещё одном терминале)
cd ../web && npm run dev
```

Откройте http://localhost:3000 — вы увидите систему!
